== Description
Easy to use jQuery plugin for zoom & pan image cropping.

== Demo
Visit: http://www.tmatthew.net/jwindowcrop

== Usage
<tt>
<img class="crop_me" src="http://placekitten.com/800/600" />

<script>
	$(function() {
		$('.crop_me').jWindowCrop({
			targetWidth:300,
			targetHeight:300
		});
	});
</script>
</tt>
